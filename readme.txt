This is the tube preamplifier project made with using what was discussed in the Graduation Thesis done by this author. 

It uses two 12AU7 tubes on each channel and can precisely match the RIAA frequency response. 

LIBRARIES:

Libs must be updated as needed, but MUST, in order to padronize, use the following descriptions:
	- Reference 
	- Value 
	- Footprint
	- Datasheet
	- OBS
